# Void Room

Void Room is an atmospheric escape puzzle game where players must navigate through mysterious environments and solve intricate puzzles to find their way out.

## Project Description

- **Game Name**: Void Room
- **Genre**: Escape / Puzzle
- **Platform**: PC (Unity)

## Getting Started

To run this project, you'll need Unity installed (the version used for development). Clone the repository and open the folder in Unity Hub.
